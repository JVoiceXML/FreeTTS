JSAPI HelloWorld! Demo

 


To run this HelloWorld! demo on a Unix system, type:

   gmake run

If you want to use MBROLA voices, please refer to the 
MBROLA Setup Page for MBROLA installation instructions.
Remember that you must set the MBROLA_DIR environment 
variable to your MBROLA directory. To use MBROLA voices, type one of:

   gmake runMbrola1
   gmake runMbrola2
   gmake runMbrola3


To run this demo on a win32 system, type:

$ run HelloWorld



It should say "Hello world!".  Click AU or WAV to listen to it.
