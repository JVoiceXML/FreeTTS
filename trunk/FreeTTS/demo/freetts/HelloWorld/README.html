FreeTTS HelloWorld! Demo

 


FreeTTS Default Voice

To run this FreeTTS HelloWorld! demo, type:

   gmake run

You will hear:

"Thank you for giving me a voice. I'm so glad to say hello to this world."

 Click AU or WAV to listen to it.

MBROLA Voices

  1. Download the MBROLA binary and voices from the MBROLA Project Homepage. Refer to MBROLA Setup Page for installation instructions.

  2. Set the MBROLA_DIR environment variable to your MBROLA directory.

  3. Type one of:

    gmake runMbrola1 (woman's voice)
    gmake runMbrola2 (man's voice)
    gmake runMbrola3 (man's voice)
