JSAPI JTime Demo

 

The JSAPI JTime demo uses a high quality (cluster unit) voice
to tell you the time.


To run this JTime demo, type:

gmake run


You should hear the current time and see the following prompt:


Enter time (HH:MM):


Enter the time in a format as follows:


Enter time (HH:MM): 12:35


You should hear "The time is now exactly twenty-five to one in 
the afternoon".  Click AU or WAV to listen to it.
